# SmallTools

A collection of free, fast, browser-based utility tools. No login, no ads, no data sent to any server.

## Live Site

[https://toolopia.github.io/smalltools](https://toolopia.github.io/smalltools)

## Tools Available

### Text
- Word Counter
- Case Converter
- Lorem Ipsum Generator
- Text Diff
- Find & Replace
- Text Reverser
- Duplicate Line Remover
- Text Sorter

### Math
- Percentage Calculator
- Unit Converter (length, weight, temp, area, volume, speed, data, time)
- Number Base Converter (binary, octal, decimal, hex)
- BMI Calculator
- Age Calculator
- Tip Calculator
- Scientific Calculator
- Loan / EMI Calculator

### Dev
- JSON Formatter & Validator
- Base64 Encode / Decode
- URL Encoder / Decoder
- Regex Tester
- HTML Escape / Unescape
- Hash Generator (SHA-1, SHA-256, SHA-384, SHA-512)
- UUID Generator
- Markdown Preview
